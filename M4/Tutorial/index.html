HTML

  1. ¿Qué es HTML?

    HTML es la abreviación de HyperText Markup Language ( Lenguaje de Marcas de Hipertexto ). Es un lenguaje que define el significado y la estructura del contenido web.
    "Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web.
    "Marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web.

  2. ¿Qué es una etiqueta y qué tipos hay?

    Son instrucciones para definir un elemento HTML específico e indican cómo mostrarlo en la página.
    Es todo elemento encerrado entre < y >. Pueden contener atributos/propiedes.
    Hay dos tipos de etiquetas:

    1. Cerradas

      Toda aquella etiqueta que necesita de otra etiqueta igual, iniciada por una /, al final del contenido que querramos poner dentro para cerrarla.

      <ETIQUETA ATTR1="VAL1" ATTR2="VAL2" ATTR>CONTENIDO</ETIQUETA>

    2. Abiertas

      Toda aquella etiqueta que no necesita de otra etiqueta igual al final. Suele ser utilizada en etiquetas que el contenido venga incrustado dentro.

      <ETIQUETA ATTR1="VAL1" ATTR2="VAL2" ATTR>

  3. ¿Por qué veo bien la web, si hay etiquetas cerradas sin cerrar?

    El navegador autocompletará cualquier etiqueta no cerrada al final del documento pudiendo, o no, mostrar la web de forma incorrecta. Toda etiqueta cerrable, ha de cerrarse. Sin excepción.

  4. Partes de un HTML

    HTML tiene 3 partes.

    1. DOCTYPE : La definición del tipo de documento.
      Aquí se le dice al programa que vaya a abrir nuestro archivo en que formanto hemos escrito nuestro código.
      <!DOCTYPE html>
      <html>
      ...
      </html>
    2. head : La información de cabecera de la pagina.
      Aquí irán todos los parametros de la pagina. Códificación, Descripción de la web, Creador, Estilo que tendran los elementos de la web, Título, etc.
      <head>
      ...
      </head>
    3. body : El cuerpo de la web.
      Aquí irán todos los elementos visibles al usuario y alguna que otro script.
      <body>
      ...
      </body>
    <!DOCTYPE html>
    <html>
    <head>
    ...
    </head>
    <body>
    ...
    </body>
    </html>

    Un atributo es todo aquello que añade información a una etiqueta, el cual puede tener valor o no.
